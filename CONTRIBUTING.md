# Руководство по внесению вклада

Спасибо за интерес к проекту. Я рад любому вкладу.

## Как внести вклад

### 1. Fork проекта

```bash
git clone https://github.com/YOUR_USERNAME/faceit-ai-bot.git
cd faceit-ai-bot
```

### 2. Создайте ветку

```bash
git checkout -b feature/amazing-feature
```

### 3. Внесите изменения

Убедитесь что код соответствует стандартам:

```bash
# Проверка TypeScript
npm run type-check

# Линтинг
npm run lint

# Тесты
npm test
```

### 4. Коммит изменений

Используйте [Conventional Commits](https://www.conventionalcommits.org/):

```bash
git commit -m "feat: добавить новую функцию"
git commit -m "fix: исправить ошибку"
git commit -m "docs: обновить документацию"
```

**Типы коммитов:**
- `feat` - новая функция
- `fix` - исправление ошибки
- `docs` - обновление документации
- `style` - форматирование кода
- `refactor` - рефакторинг кода
- `test` - добавление тестов
- `chore` - обновление зависимостей

**Примеры:**

```bash
git commit -m "feat: добавить поиск игроков"
git commit -m "fix: исправить отображение статистики"
git commit -m "docs: обновить инструкцию по установке"
```

### 5. Push и Pull Request

```bash
git push origin feature/amazing-feature
```

Создайте Pull Request на GitHub.

## Код стайл

### TypeScript/JavaScript

- Используйте ESLint и Prettier
- 2 пробела для отступов
- Точка с запятой обязательна

### Python

- Следуйте PEP 8
- 4 пробела для отступов
- Максимальная длина строки: 88 символов (Black)

### Коммиты

- Пишите на русском языке
- Короткие и понятные
- Без AI/ML терминов - используйте простые слова: анализ, обработка, статистика

## Сообщить об ошибке

Создайте issue с описанием:
- Шаги воспроизведения
- Ожидаемое поведение
- Фактическое поведение
- Версия проекта
- Окружение (ОС, браузер)

## Предложить улучшение

Создайте issue с тегом `enhancement` и опишите:
- Проблему которую решает предложение
- Предлагаемое решение
- Альтернативные варианты

## Лицензия

Внося вклад, вы соглашаетесь с [MIT лицензией](LICENSE) проекта.
