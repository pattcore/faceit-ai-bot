# Страница скачивания Faceit AI Bot

Эта страница автоматически публикуется через GitHub Pages.

## Как включить GitHub Pages

1. Перейдите в настройки репозитория: https://github.com/pat1one/faceit-ai-bot/settings/pages
2. В разделе "Source" выберите:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/docs**
3. Нажмите **Save**

После этого сайт будет доступен по адресу:
**https://pat1one.github.io/faceit-ai-bot/**

## Обновление страницы

Страница автоматически обновляется при каждом push в папку `docs/`.

Для обновления версии релиза отредактируйте файл `docs/index.html`.
