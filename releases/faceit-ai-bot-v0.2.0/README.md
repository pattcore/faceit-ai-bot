# Faceit AI Bot 

Инструмент для анализа игровой статистики и поиска тиммейтов на платформе Faceit.

## Скачать

Вы можете скачать расширение для браузера или полный релиз на нашем сайте:

**[Скачать на официальном сайте](https://pat1one.github.io/faceit-ai-bot/)**

Доступно:
- **Расширение для браузера** - легкий способ интеграции с Faceit
- **Полный релиз (Docker)** - для локального развертывания всех сервисов

## Быстрый старт

### Установка и запуск

```bash
# 1. Установить зависимости
make install
# или: npm install

# 2. Настроить окружение
cp .env.example .env
# Отредактируйте .env файл

# 3. Собрать проект
make build
# или: ./build-all.sh

# 4. Запустить все сервисы
make deploy
# или: ./deploy.sh
```

### Доступные сервисы

После запуска:
- Frontend: http://localhost:3000
- Backend API: http://localhost:8000
- API Docs: http://localhost:8000/docs
- PostgreSQL: localhost:5432

```bash

make deploy        # Деплой всех сервисов

make clean         # Очистить временные файлы
```

## Структура проекта

- `app/` - Next.js приложение
- `src/` - Исходный код (backend + browser extension)
- `public/` - Статические файлы
- `Dockerfile.*` - Docker конфигурации
- `docker-compose.yml` - Оркестрация сервисов

## Технологии

- **Frontend**: Next.js 15, React 19, TypeScript
- **Backend**: FastAPI, Python 3.9+
- **Database**: PostgreSQL 16
- **Browser Extension**: Webpack, Babel
- **Deployment**: Docker, Docker Compose

## Возможности

### Анализ статистики
- Просмотр детальной статистики матчей
- Отслеживание личного прогресса
- Сравнение показателей с другими игроками

### Поиск тиммейтов
- Фильтрация игроков по критериям
- Просмотр статистики потенциальных тиммейтов
- Удобный интерфейс для коммуникации

### Уведомления
- Информация о новых матчах
- Обновления статистики
- Интеграция с профилем Faceit

## Как использовать

1. **Установите расширение** или разверните полную версию
2. **Авторизуйтесь** через Faceit
3. **Начните анализировать** свою статистику и искать тиммейтов
